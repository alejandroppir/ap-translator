Click on one of this menus to start

This is a web app for translating into multiple languages fast.
First you need to configure the languages.
To do this it is necessary to go to the menu on the right (gear) and add the languages
Language code is the 2 letters identifier for the language. You can check them here.
The language description is the name that you want to assign to the language.
And the translation icon will identify which one is the source language If you like the app you could give me a star on github star .
